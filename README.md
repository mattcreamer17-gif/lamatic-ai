# Lamatic.ai landing page

A single-file static page prepared for Lamatic.ai (Fort Lauderdale, GenAI platform, 18 people, ~$1M ARR). Tag is FIT-PARTNERSHIP, so the page is written peer-to-peer for CEO Charles Whiteman and CTO Aman Sharma rather than as a consulting sale. The pitch reflows fit-analysis section 7: the 60/30/10 authoring frame, a scoped-sprint case from Feeld, the ICM paper, and a calendar link to Matt Creamer. Design tokens pulled from Lamatic's live Webflow CSS: Inter Tight, near-white canvas (#f9f9f9), warm red accent (#f33736) against a black product block that echoes their node-graph screenshot aesthetic. Mobile responsive at 375px. Footer is the only place Eduba's name appears.
